요소의 여백과 관련된 속성

        HTML에서 요소가 가지는 영역은 4가지로 구성된다.
        
        1. content 영역
         - 요소의 내용이 작성되는 영역
         - 시작태그와 종료태그 사이의 값이 저장된다.
        2. padding 영역
         - 요소의 여백을 표현하는 영역역
         - content 영역과 border 영역 사이에 존재한다.
        3. border 영역
         - 요소의 테두리가 지정되는 영역
         - content 영역을 감싸는 역할을 한다.
        4. margin 영역
         - 현재 요소와 다른 요소의 간격을 나타내는 영역역

        화면에 보여지는 부분인 content, padding, border는
