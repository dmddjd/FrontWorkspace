1. 속성 선택자

스타일을 부여하고자 하는 요소 내에 작성되어있는 속성을 이용하여 요소를 선택하는 선택자
선택자 뒤에 []를 이용하여 속성과 속성값을 제시 

        1. 기본선택자[속성=속성값] = 일치
        2. 기본선택자[속성~=속성값] = 포함
        3. 기본선택자[속성|=속성값] = -기준 앞 단어
        4. 기본선택자[속성^=속성값] = 시작값
        5. 기본선택자[속성$=속성값] = 끝 값
        6. 기본선택자[속성*=문자] = 문자가 포함된 태그

2. 자손/후손 선택자

자손 : 상위요소의 바로 아래에 존재하는 요소
후손 : 상위요소의 아래에 존재하는 모든 요소 

        *자손 선택자
        a>b{속성:속성값}

        *후손 선택자
        a b{속성:속성값}

  • ul태그의 자손/후손, div태그의 후손
  • ul태그의 자손/후손, div태그의 후손

3. 동위레벨 선택자

동위관계에 있는 특정 요소를 선택할 때 사용하는 선택자

        *a요소 뒤에있는 b요소 하나를 선택하는 선택자
        a+b{속성:속성값;}
        
        *a요소 뒤에있는 b요소 전체를 선택하는 선택자
        a~b{속성:속성값;}

4. 반응 선택자

사용자의 움직임에 따라 요소를 선택하는 선택자

        *요소가 클릭될 경우 요소를 선택하는 선택자
        선택자:active{속성:속성값}

        *요소에 마우스가 올라갈 경우 요소를 선택하는 선택자
        선택자:hover{속성:속성값}

5. 상태 선택자

요소의 상태에 따라 선택되는 선택자

        1) checked(선택된) 상태의 요소를 선택하는 선택자
    
        2) focus(초점)이 맞춰진 input요소만 선택하는 선택자
        선택자:focus{속성:속성값}
