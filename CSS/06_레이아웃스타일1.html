        사전적 의미 : 배치
        기술적 의미 : 구성 요소들을 제한된공간에 배치하는 기술

화면 배치 방법 : display

        요소가 화명네 어떻게 보여질지 지정하는 속성

        1. block : 수직 분할 (행 분할) 
        2. inline : 수평 분할 (열 분할)
        3. inline-block : 행 분할 + 열 분할
        4. none : 요소를 보여주지 않음
         - display: none : 요소는 존재하지만 화면상에는 보이지 않게 하는 속성
         - opacity: ?% : 요소의 투명도를 조절하는 속성
         - visibility: hidden : 요소를 투명하게 하는 속성
