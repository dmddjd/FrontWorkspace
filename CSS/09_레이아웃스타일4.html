배치 관련 속성(position)

        1. position: relative
            - 하위 요소들의 기준으로 지정하는 속성
            - 하위 요소들은 top, bottom, left, right 속성을 사용할 수 있다.
        
        2. position: absolute
            - 기존 요소들의 배치 순서를 무시하고 지정된 위치에 배치하는 속성
        
        3. position: fixed
            - 요소를 항상 고정된 뷰포트 위치에 배치하는 속성
