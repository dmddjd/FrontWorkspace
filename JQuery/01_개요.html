jQuery개요

jQuery란?

기존의 JS언어로만 구현하기에는 복잡한 구문 작성들을
간소화하기 위해 개발된 JS기반 라이브러리이다.
즉, JS방식으로 작성된 유용한 함수 및 메서드들이 정의되어있다.

복잡한 JS구문을 보다 짧고 쉽게 개발할 수 있어 생산성이 크게 향상된다.

DOM요소와 관련된 조작기능, 이벤트 처리기능,
AJAX기능, 추가 확장 플러그인등 다양한 기능을 제공한다.

jQuery 구문 작성

    
jQuery구문으로 DOM요소를 다루는 구문을 작성할 예정이며,
jQuery명령어들은 문서상의 모든 html요소들이 로드된 후에 실행되어야 한다.
    
        
    1. 순수 JS로 html페이지가 모두 로드된 후에 실행되게 하려면 
       - window.onload = function(){
              // 실행 내용
         }

    2. jQuery에서의 onload함수
       - jQuery(document).ready(function(){
              // 실행내용
         });

       - $(document).ready(function(){
              // 실행내용
         });

    3. $(function(){
            // 실행내용
       });
