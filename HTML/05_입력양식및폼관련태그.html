폼 태그 관련

form태그 내의 submit버튼 클릭 시 해당 form내에 작성된 사용자가 입력한 값들이 서버에 전달된다.
[표현법]
form action="속성값" method="속성값"
*action 속성 : form태그 내부의 입력한 값들을 전달함
"서버경로"
*method 속성 : 서버로 전달시 전송방식을 지정하는 속성
> get / post 두 가지 방식이 존재
1. get : 요청시 사용자가 전달한 값들이 URL에 노출되는 방식
2. post : 요청시 사용자가 전달한 값들이 URL에 노출되지 않는 방식

라디오버튼과 체크박스

라디오버튼

제시한 여러개의 값들 중 오직 1개만 선택해야하는 경우 사용
서버로 요청시 선택된(check) 체크박스의 value값이 전송된다
name 속성값이 동일한 radio버튼끼리 하나의 그룹이 만들어진다

체크박스

제시한 여러개의 값들 중에서 여러개를 선택해야 하는 경우 사용
현재 선택된 체크박스의 value값들이 서버로 전송된다. name속성값을 통일시켜 하나의 그룹으로 만들어줘야 한다.

select태그와 obtion태그

- 사용자에게 직접 키보드로 값을 입력받는게 아니라
내가 제시한 여러 옵션들 중 하나를 선택할 수 있게 하는 태그
- 서버로 요청시 현재 선택된(select)값이 서버로 넘어간다.
- 각 option태그에 value속성값을 명시한 경우 value값이 서버로 넘어가고,
명시하지 않은 경우 content영역의 값이 서버로 전송된다.
